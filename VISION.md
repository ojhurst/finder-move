# Finder Move — Vision

## What This Is
A macOS Finder extension that adds "Move Item Here (Paste)" to your right-click menu.

## The Problem
macOS can move files instead of copying them with Cmd+Option+V, but nobody remembers that shortcut. It should just be in the context menu.

## What It Does Today
- Adds a "Move Item Here (Paste)" option to the Finder right-click menu
- Copy a file, navigate to the destination, right-click, move. Done.
- Handles name conflicts with a replace/skip prompt
- Clears the clipboard after the move

## Where It's Going
This is basically done. It does one thing and does it well. If Apple ever adds this to Finder natively, this becomes unnecessary. Until then, it fills the gap.

The context menu item that should have been there from the start.
