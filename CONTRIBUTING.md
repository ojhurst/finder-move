# Contributing to Move Here for Finder

Thanks for wanting to help! This is a small project, so the process is simple.

## Getting started

1. Fork the repo
2. Clone your fork
3. Make sure you have [XcodeGen](https://github.com/yonaskolb/XcodeGen) installed: `brew install xcodegen`
4. Run `xcodegen generate` to create the Xcode project
5. Build and test with Xcode or `xcodebuild`

## Making changes

- Create a branch off `main` for your work
- Keep changes focused — one feature or fix per PR
- Test on your own machine before submitting (copy a file, right-click, make sure it moves)

## Submitting a pull request

1. Push your branch to your fork
2. Open a PR against `main`
3. Describe what you changed and why

## Ideas for contributions

- Pre-built `.app` releases so people don't need Xcode
- Launch-at-login support
- Localization (menu item text in other languages)
- Badge overlay on copied files to show they're "ready to move"
- Keyboard shortcut support

## Code style

Nothing fancy. Keep it readable. Match what's already there.
